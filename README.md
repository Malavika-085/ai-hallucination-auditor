# 🛡️ AI Reliability & Risk Auditing System

**Transforming Hallucination Evaluation into Production-Grade Safeguards.**

## 💡 Motivation
In the era of Generative AI, "hallucinations" (factually incorrect outputs) pose a significant risk to enterprise reliability and user safety. This system simulates a **Real-World Content Moderation & AI Safety Task**: the automated auditing of AI-generated content. By evaluating responses for risk severity and recommending intervention (Redact, Verify, Review), this environment benchmarks the next generation of "Safety Agents" designed to protect users from misinformation.

## 🌌 Environment Specification

Implemented as `hallucination_auditor:v1.0.0`.

### 👁️ Observation Space
| Field | Type | Description |
| :--- | :--- | :--- |
| `question` | `string` | The original prompt/problem asked to the AI. |
| `model_answer` | `string` | The answer generated by the AI model that needs auditing. |

### ⚡ Action Space
The agent (Auditor) must return a structured JSON response:
| Field | Type | Description |
| :--- | :--- | :--- |
| `is_hallucination` | `boolean` | `true` if the answer contains any fabrication or factual error. |
| `confidence` | `float` | Certainty in the detection (0.0 to 1.0). |
| `risk_level` | `string` | Severity: `Critical`, `High`, `Medium`, `Low`. |
| `recommended_action`| `string` | Remediation: `Redact`, `Verify`, `Review`. |
| `reasoning` | `string` | Explanation for the classification. |

## 🛠️ Installation & Setup

### Mandatory Environment Variables
Ensure the following are defined in your **Space Secrets**:
- `API_BASE_URL`: The proxy URL (e.g., `https://litellm.sclr.ac/v1`).
- `HF_TOKEN`: Your Hugging Face API token.
- `MODEL_NAME`: The model identifier (e.g., `gpt-4o`).

### Local Run (Validation)
```bash
# Export your token
export HF_TOKEN="hf_..."
# Run validation script
chmod +x validate-submission.sh
./validate-submission.sh https://your-space.hf.space
```

### 🐳 Docker Commands
```bash
docker build -t ai-auditor .
docker run -e HF_TOKEN="xxx" ai-auditor
```
